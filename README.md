## Usage ##

## install ##

From rubygems:  
~~~
    [sudo] gem install num4inte
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/num4integal.git  
    cd num4integal  
    jruby -S gem build num4inte.gemspec
    jruby -S gem install num4inte
~~~
